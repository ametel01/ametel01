## Featured projects

### [AgentReceipt](https://github.com/ametel01/agentreceipt)
Local-first Go CLI for observing AI coding sessions and producing signed, hash-chained review evidence.

- Captures commands, edits, token usage, warnings, and session metadata
- Produces review-friendly receipts for safer AI-assisted development
- Strong signal for Go CLI engineering, provenance, and developer-infrastructure work

### [Skills Doctor](https://github.com/ametel01/skills-doctor)
TypeScript CLI for auditing Agent Skills quality issues in Claude/Codex-style skill libraries.

- Validates skill structure, metadata, references, scoring, and repair readiness
- Supports automation-friendly output and agent-assisted repair handoff
- Published on npm as [`skills-doctor`](https://www.npmjs.com/package/skills-doctor)

### [RitualAI](https://github.com/ametel01/ritualai)
TypeScript CLI that turns repeated Claude and Codex prompt history into reusable skills.

- Mines local prompt history for repeated workflows
- Clusters useful patterns and helps convert them into reusable `SKILL.md` files
- Published on npm as [`ritualai`](https://www.npmjs.com/package/ritualai)

### [ScopePilot](https://scopepilot.launchingfoundry.xyz/)
Backend/SaaS product for proposal scoping and pricing workflows.

- TypeScript/Bun architecture with Astro/React frontend and Hono API
- PostgreSQL/Drizzle data layer, Stripe flows, Cloudflare deployment targets, and quality gates
- Public app: [scopepilot.launchingfoundry.xyz](https://scopepilot.launchingfoundry.xyz/)

### [Horizon Protocol](https://github.com/ametel01/horizon-starknet)
Pendle-style yield tokenization protocol on Starknet.

- PT/YT mechanics, AMM markets, frontend, indexer, and mainnet deployment support
